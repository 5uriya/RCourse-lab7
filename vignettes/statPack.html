Linear regression function: linreg()

It take 2 parameter as input

‘linreg(formula, data)’

  1. formula (for example: formula = Petal.Length ~ Species) left is dependent variable stored in variable in Y and right is independent Variable which is stored as matrix in X
  2. data a default data frame is load from R (for example data = iris or faithful)

Calculate the following different linear algebra functions:

  1. Regression coefficients (betaCoff) : \(\hat{\beta}= (X^TX)^{-1}X^Ty\)

  2. The fitted values (fittedValues) : \(\hat{y}= X\hat{\beta}\)

  3. The residuals (residualsVal) : \(\hat{e}= y-X\hat{\beta}\)

  4. The degree of freedom (degree_f) : \(df=n-p\) , where \(n\) is the number of observations and \(p\) is the number of parameters in the model

  5. The residual variance (residualsVariance) : \(\hat{\sigma}^2=\frac{e^Te}{df}\)

  6. The variance of the regression coefficients (regression_coff) : $()=2(XTX)^{-1}) $

  7. The t-values for each coefficient (tValues) :\(t_\beta=\frac{\hat{\beta}}{\sqrt{Var(\hat{\beta})}}\)

  8. The p-values for each coefficient (pValue) : \(P(|>t|)\) the probability of each coefficient to be equal to 0
